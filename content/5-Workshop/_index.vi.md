# Guide to Building an Online Ride-Booking System (Serverless)

#### Overview

Welcome to the Workshop on building a **Serverless Car Booking** system.
In this section, we will step by step deploy a complete ride-booking system using AWS's Serverless ecosystem, including: AWS Amplify, Amazon API Gateway, AWS Lambda, Amazon DynamoDB, and Amazon Cognito.

This workshop is designed to give you a practical look at how AWS services connect with one another to solve a real-world e-commerce problem, while ensuring security, automatic scalability, and cost optimization (Pay-as-you-go).
