# Tuần 3 - Amazon EC2 và Dịch vụ Compute

### Mục tiêu tuần 3:

* Tìm hiểu kiến thức cơ bản về điện toán đám mây với Amazon EC2.
* Học cách khởi tạo, cấu hình và quản lý EC2 Instance.
* Tìm hiểu Amazon Machine Image (AMI), Instance Type và Amazon EBS.
* Hiểu cách Security Group kiểm soát lưu lượng mạng đến EC2 Instance.
* Thực hành kết nối đến EC2 thông qua SSH.
* Học cách quản lý vòng đời tài nguyên EC2 và tối ưu việc sử dụng tài nguyên.

### Kết quả đạt được trong tuần 3:

* Hiểu rõ hơn về Amazon EC2 và các tài nguyên điện toán trên nền tảng Cloud.

* Tìm hiểu các khái niệm chính của Amazon EC2, bao gồm:
  * EC2 Instance
  * Instance Type
  * AMI
  * EBS
  * Security Group
  * Key Pair
  * Elastic IP

* Biết cách lựa chọn EC2 Instance Type phù hợp với nhu cầu sử dụng.

* Khởi tạo và cấu hình thành công EC2 Instance.

* Cấu hình Security Group để kiểm soát lưu lượng mạng Inbound và Outbound.

* Tạo và sử dụng thành công EC2 Key Pair.

* Kết nối thành công đến EC2 Instance thông qua SSH.

* Thực hành tạo, Attach và Detach EBS Volume.

* Tìm hiểu cách tạo EBS Snapshot để sao lưu dữ liệu.

* Thực hành quản lý vòng đời của EC2 Instance:
  * Start
  * Stop
  * Restart
  * Terminate

* Cải thiện kỹ năng thực hành với dịch vụ Compute và quản lý hạ tầng Cloud cơ bản trên AWS.
