# Week 2 - AWS IAM and Amazon VPC

### Week 2 Objectives:

* Understand AWS Identity and Access Management (IAM).
* Learn how to manage users, groups, roles, and policies.
* Understand authentication, authorization, and the Principle of Least Privilege.
* Learn the fundamentals of Amazon Virtual Private Cloud (VPC).
* Understand basic AWS networking concepts.
* Practice creating and configuring basic AWS security and networking resources.
