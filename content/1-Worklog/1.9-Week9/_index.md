# Week 9 - Data Lake and Data Analytics on AWS

### Week 9 Objectives:

* Understand the fundamentals of data analytics on AWS.
* Learn the concept and architecture of a Data Lake.
* Understand how Amazon S3 can be used as a data storage layer.
* Learn the basic concepts of AWS Glue and data cataloging.
* Use Amazon Athena to query data stored in Amazon S3.
* Explore Amazon QuickSight for data visualization.
* Understand the basic data processing and analytics workflow on AWS.
