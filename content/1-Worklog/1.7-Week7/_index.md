# Week 7 - AWS IAM, Systems Manager and Infrastructure as Code

### Week 7 Objectives:

* Understand AWS Identity and Access Management (IAM).
* Learn the Principle of Least Privilege.
* Understand IAM Users, Groups, Roles, and Policies.
* Learn how applications can access AWS services using IAM Roles.
* Explore AWS Systems Manager for managing cloud resources.
* Understand Infrastructure as Code with AWS CloudFormation.
* Practice deploying AWS resources using CloudFormation templates.
