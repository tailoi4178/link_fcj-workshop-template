# Week 1 - Exploring AWS Fundamentals

### Week 1 Objectives:

* Get familiar with the First Cloud AI Journey (FCAJ) internship program and working environment.
* Understand fundamental concepts of Cloud Computing and Amazon Web Services (AWS).
* Learn about AWS global infrastructure and the main categories of AWS services.
* Create and configure an AWS account and become familiar with the AWS Management Console.
* Learn basic AWS CLI usage and understand fundamental cloud computing concepts.
