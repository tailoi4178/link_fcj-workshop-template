# Week 6 - Monitoring with Amazon CloudWatch and DNS with Amazon Route 53

### Week 6 Objectives:

* Understand the fundamentals of monitoring and observability on AWS.
* Learn how Amazon CloudWatch collects logs and metrics.
* Create CloudWatch Dashboards and Alarms.
* Learn how to monitor EC2 resource utilization.
* Understand DNS concepts and Amazon Route 53.
* Learn how to configure DNS records and routing policies.
* Practice using AWS CLI to manage and monitor AWS resources.
